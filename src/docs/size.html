Size

Toutes les tailles (espaces, textes et composants) sont gérées à partir des tailles des fonts définies dans typography.css. Pour profiter de ce système, l'unité de référence devra être em pour les espaces (padding, margin, border).

La taille de base est laissé à l’appréciation de l'internaute en laissant un valeur de 100% sur l'élément html. Les tailles utilisent un ratio unique entre elles, ce qui permet de réduire ou agrandir la taille de tous les élément d'un composant vers la taille supérieur ou inférieur.

Les modificateurs de tailles .xsmall, .small, .medium, .large, .xlarge ne modifient que la valeur de font-size.
Les modificateurs de tailles .h6 -> .h1 modifient font-size et line-height de manière à reproduire fidèlement les headers équivalemment.
