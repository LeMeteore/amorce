Grid

3 systèmes de grilles sont disponibles :

  • .tb-grid Basée sur le table layout.
  • .fl-grid Basée sur un layout en float left.
  • .il-grid Basée sur un layout en inline-grid.

Le system de grille permet de composer avec un système de 2 à 7 colonnes imbricables, avec ou sans gouttières.
Pour supprimer les gouttières à un système de grille, simplement ajouter la class no-gutter sur le wrapper de la grille.
